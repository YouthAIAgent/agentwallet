The Problem

AI agents are becoming autonomous economic actors. They negotiate, trade, and pay for services. But they're operating with human wallets, zero programmatic control, no spending limits, no audit trail.

  • No per-agent financial identity
  • No spending limit enforcement
  • No escrow for task payments
  • No compliance or audit trail
✓

The Solution

AgentWallet gives every AI agent its own on-chain identity with programmable spending limits, trustless escrow, real-time analytics, and full compliance.

  • Dedicated wallet per agent (KMS-encrypted)
  • JSON policy engine (per-tx + daily limits)
  • On-chain escrow (Anchor PDAs)
  • Immutable audit log + anomaly detection

Everything agents need to move money

🔑

Custodial Wallets

Dedicated Solana wallets per agent. Private keys encrypted at rest with Fernet/AWS KMS. Keys never leave the server.

🛡

Spending Policies

JSON-based policy engine. Per-transaction limits, daily caps, destination whitelists, time windows, human approval workflows.

🔒

Trustless Escrow

On-chain Anchor PDAs hold funds. Create, release, refund, dispute. Auto-expire stale escrows. No one can rug.

Real-time Analytics

Pre-aggregated daily rollups. Per-agent cost breakdowns, spending trends, ROI tracking. CSV export.

Python SDK

Compliance & Audit

Ship in 5 lines of code

quickstart.py 
from agentwallet import AgentWallet

async with AgentWallet(api_key="aw_live_...") as aw:

    # Spawn agent with its own wallet
    agent = await aw.agents.create(
        name="trading-bot",
        capabilities=["trading", "payments"]
    )

    # Send SOL (policy-enforced, fee-deducted, audit-logged)
    tx = await aw.transactions.transfer_sol(
        from_wallet=agent.default_wallet_id,
        to_address="RecipientPubkey...",
        amount_sol=0.5,
    )

    # Lock funds in escrow for a task
    escrow = await aw.escrow.create(
        funder_wallet=agent.default_wallet_id,
        recipient_address="WorkerAgent...",
        amount_sol=1.0,
        expires_in_hours=24,
    )
    await aw.escrow.release(escrow.id)
